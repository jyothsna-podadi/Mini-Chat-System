# Python Chat Room System

A simple Object-Oriented Programming (OOP) based chat room simulation built in Python.

## Features

* Multiple users can join a chat room
* Send messages to the chat room
* Automatic message IDs
* Chat history storage
* Join and leave functionality
* OOP concepts implementation

## Technologies Used

* Python 3
